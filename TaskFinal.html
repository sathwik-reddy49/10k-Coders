HTML Questions and Answers

1. What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating webpages.

2. Why do we use HTML?

HTML is used to create and structure content on the web.

3. What is <!DOCTYPE> in HTML?

<!DOCTYPE> is a declaration that defines the document type and version of HTML being used.

4. What is the root element of HTML?

The root element of an HTML document is the <html> tag.

5. What is a tag?

A tag is a markup construct that defines an HTML element.

6. Give me a few examples of self-closing tags?

Self-closing tags are HTML elements that do not require a closing tag, such as:

  • <img src="image.jpg" alt="Description">
  • <br>
  • <hr>
7. What is an element?

An element is a complete HTML structure, consisting of a start tag, content, and an end tag.

8. How many types of elements are there in HTML?

There are two main types of elements in HTML: block elements and inline elements.

9. What is the difference between inline vs block elements?

Block elements occupy the full width and start on a new line, while inline elements only take up as much space as their content.

10. What is the difference between semantic elements and non-semantic elements?

Semantic elements clearly describe their meaning, while non-semantic elements do not convey specific meaning.

11. What are href vs src attributes?

href is used for links, while src is used for images or scripts.

12. What are the features of HTML5?

HTML5 introduces new elements, attributes, and APIs for improved functionality, including:

  • New semantic elements (<article>, <section>, <header>, <footer>)
  • Multimedia support (<audio>, <video>)
  • Local storage capabilities
  • Improved form controls
13. What are entities in HTML?

Entities are special characters represented by a string of characters. For example, to display <, >, and & you use: &lt; &gt; &amp;.

14. What is <div> vs <span>?

<div> is a block-level element, while <span> is an inline element.

15. What are list tags in HTML?

List tags are used to create ordered and unordered lists. Examples include:

  • <ul> for unordered lists
  • <ol> for ordered lists
  • <li> for list items
16. What are attributes?

Attributes provide additional information about an HTML element and are defined within the opening tag.

17. Mention a few attributes for <img> tags?

Common attributes for <img> tags include:

  • src: URL of the image
  • alt: Alternative text for the image
  • width: Width of the image
  • height: Height of the image
18. What is <thead> on the table?

<thead> is an element used to group the header content in an HTML table.

19. What is <tr>?

<tr> stands for table row and is used to define a row of cells in a table.

20. What is <th>?

<th> is a table header cell, which is typically bold and centered, used to define a header for a column or row in a table.

21. What is <td>?

<td> is a table data cell, which contains data for a specific row and column in a table.

22. Can I use <th> in <tbody>?

Yes, you can use <th> within <tbody> to define header cells for specific rows, but it is typically used in <thead>.

23. What are meta tags?

Meta tags provide metadata about the HTML document, such as character set, description, keywords, author, and viewport settings.

24. What is SEO?

SEO (Search Engine Optimization) is the practice of optimizing web content to improve its visibility in search engine results.

25. What is the attribute of the HTML tag?

The <html> tag can have attributes like lang (to specify the language) and dir (to specify text direction).

26. What is an iframe and anchor?

Iframe: The <iframe> element allows you to embed another HTML document within the current one. It's often used for embedding videos, maps, or external content. Security concerns, such as cross-origin resource sharing (CORS), may arise depending on the content source.

Anchor (<a>): The <a> tag creates hyperlinks, allowing users to navigate between pages. Attributes like target="_blank" are often used for external links.

27. What is Git vs GitHub?

Git: Git is a version control system that tracks changes in source code, allowing multiple people to collaborate on a project efficiently. It operates locally on your machine.

GitHub: GitHub is a web-based platform for hosting Git repositories, offering collaboration tools, pull requests, issue tracking, and more.

28. What is the workflow of GitHub?

The general GitHub workflow includes:
1. Cloning a repository.
2. Making changes locally.
3. Committing changes.
4. Pushing changes to the remote repository.
5. Opening pull requests for code review.
6. Merging changes after review.

29. What command do we need to use to move the changes to staged changes?

The git add command stages changes. To stage a specific file: git add filename. To stage all files: git add .

30. How can you stage all files at a time?

Use the command git add . to stage all modified and newly created files at once.

31. What is the git push command and use of it?

The git push command is used to upload local repository content to a remote repository. For example, git push origin main pushes your changes to the main branch on the remote repository.

32. What is git commit command usage?

The git commit command saves staged changes as a new snapshot in the repository's history. Use git commit -m "message" to commit with a message. The git commit --amend can be used to modify the last commit.

33. What is the use of the git init command?

The git init command initializes a new Git repository in a directory. It's the first step when starting version control for a project.

34. What are formatting tags?

Formatting tags in HTML include <b>, <i>, <u>, <mark>, etc., used for styling text. These tags are non-semantic and primarily for presentation.

35. What are input tag attributes?

Attributes for the <input> tag include:
- type (specifies input type: text, password, checkbox, etc.)
- name (used to reference form data)
- placeholder (displays placeholder text)
- required (marks input as mandatory)
- value (sets default value).

36. What are form attributes?

Key form attributes include:
- action (specifies where to send form data)
- method (defines how form data is sent: GET or POST)
- enctype (specifies encoding for form data, used for file uploads).

37. What is a marquee tag?

The <marquee> tag creates a scrolling text or image. It's deprecated in HTML5 but still used in some legacy code.

38. What are marquee tag attributes?

Attributes include: direction (sets scroll direction), behavior (scroll, slide, alternate), scrollamount (speed), and loop (number of iterations).

39. What are multimedia tags?

Multimedia tags include <audio>, <video>, and <embed>. These tags allow you to embed media content (sound, video, and other objects) in web pages.

40. Can we use block elements in inline elements?

Technically, block elements should not be nested inside inline elements as this violates HTML structure rules. However, modern browsers may still render such content.

41. What are differences between absolute vs relative paths?

Absolute paths point to a specific location on the server and are used for external links (e.g., https://example.com/image.jpg).

Relative paths are relative to the current directory and used for internal resources (e.g., ./images/pic.jpg).

42. What are summary and details tags?

The <summary> and <details> tags are interactive elements that allow content to be collapsed or expanded. It's useful for creating FAQ sections or hiding large blocks of information by default.

43. What tag is helpful to give the reviews?

The <blockquote> tag is often used for displaying reviews or quotes from other sources, giving them proper semantic meaning.

44. What are the fieldset and legend tags?

<fieldset> groups form elements together, making them easier to manage and style. <legend> provides a caption for the <fieldset>, offering context or instructions for the grouped elements.
