HTML Questions and Answers

1. What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating webpages.

2. Why do we use HTML?

HTML is used to create and structure content on the web.

3. What is <!DOCTYPE> in HTML?

<!DOCTYPE> is a declaration that defines the document type and version of HTML being used.

4. What is the root element of HTML?

The root element of an HTML document is the <html> tag.

5. What is a tag?

A tag is a markup construct that defines an HTML element.

6. Give me a few examples of self-closing tags?

Self-closing tags are HTML elements that do not require a closing tag, such as:

  • <img src="image.jpg" alt="Description">
  • <br>
  • <hr>
7. What is an element?

An element is a complete HTML structure, consisting of a start tag, content, and an end tag.

8. How many types of elements are there in HTML?

There are two main types of elements in HTML: block elements and inline elements.

9. What is the difference between inline vs block elements?

Block elements occupy the full width and start on a new line, while inline elements only take up as much space as their content.

10. What is the difference between semantic elements and non-semantic elements?

Semantic elements clearly describe their meaning, while non-semantic elements do not convey specific meaning.

11. What are href vs src attributes?

href is used for links, while src is used for images or scripts.

12. What are the features of HTML5?

HTML5 introduces new elements, attributes, and APIs for improved functionality, including:

  • New semantic elements (<article>, <section>, <header>, <footer>)
  • Multimedia support (<audio>, <video>)
  • Local storage capabilities
  • Improved form controls
13. What are entities in HTML?

Entities are special characters represented by a string of characters. For example, to display <, >, and & you use: &lt; &gt; &amp;.

14. What is <div> vs <span>?

<div> is a block-level element, while <span> is an inline element.

15. What are list tags in HTML?

List tags are used to create ordered and unordered lists. Examples include:

  • <ul> for unordered lists
  • <ol> for ordered lists
  • <li> for list items
16. What are attributes?

Attributes provide additional information about an HTML element and are defined within the opening tag.

17. Mention a few attributes for <img> tags?

Common attributes for <img> tags include:

  • src: URL of the image
  • alt: Alternative text for the image
  • width: Width of the image
  • height: Height of the image
18. What is <thead> on the table?

<thead> is an element used to group the header content in an HTML table.

19. What is <tr>?

<tr> stands for table row and is used to define a row of cells in a table.

20. What is <th>?

<th> is a table header cell, which is typically bold and centered, used to define a header for a column or row in a table.

21. What is <td>?

<td> is a table data cell, which contains data for a specific row and column in a table.

22. Can I use <th> in <tbody>?

Yes, you can use <th> within <tbody> to define header cells for specific rows, but it is typically used in <thead>.

23. What are meta tags?

Meta tags provide metadata about the HTML document, such as character set, description, keywords, author, and viewport settings.

24. What is SEO?

SEO (Search Engine Optimization) is the practice of optimizing web content to improve its visibility in search engine results.

25. What is the attribute of the HTML tag?

The <html> tag can have attributes like lang (to specify the language) and dir (to specify text direction).
